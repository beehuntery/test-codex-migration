# Tasks UI 概要

このページは Next.js フェーズ3で構築中のタスク管理 UI コンポーネントを Storybook 上で解説するための MDX ドキュメントです。

## TaskCard コンポーネント

タスクのタイトル、説明、期限、タグ、ステータスを表示するカードです。Next.js ではサーバーコンポーネントから呼び出され、ステータス更新フォーム（Server Action）を組み合わせることで操作が可能になります。

## 確認ポイント

- Tailwind トークンによるスタイリングとデザイントークンの適用状況
- タグがない場合のプレースホルダーテキスト
- Server Action を利用できない環境でもレンダリングが崩れないか

## TaskDeleteButton コンポーネント

タスク削除操作を行うためのクライアントコンポーネントです。確認ダイアログを内包しており、Server Action を通じて削除リクエストを発行します。
