# Tasks UI 概要

このページは Next.js フェーズ3で構築中のタスク管理 UI コンポーネントを Storybook 上で解説するための MDX ドキュメントです。

## TaskCard コンポーネント

タスクのタイトル、説明、期限、タグ、ステータスを表示するカードです。Next.js ではサーバーコンポーネントから呼び出され、ステータス更新フォーム（Server Action）を組み合わせることで操作が可能になります。
